# PARTIE 1 — VISION & PRODUIT

## Le pitch en une phrase
"Filme ton produit 60 secondes avec ton iPhone. Tes clients le voient en vrai chez eux — sans télécharger d'app."

## Ce que c'est
ARShot est un SaaS B2B multilingue (FR/EN/ES/DE) qui permet aux PME et e-commerçants de créer des expériences AR pour leurs produits, sans aucune compétence technique.

## Le produit complet — 4 pilliers

```
PILIER 1 — GÉNÉRATION 3D
  iPhone filme 60s → modèle 3D photoréaliste automatique
  QR code + lien shareable en 3 minutes

PILIER 2 — ARSHOT STUDIO
  1 bouton → vidéos virales TikTok/Reels/Shorts prêtes à poster
  Trend Sync, POV Generator, Auto-Post, A/B Test Engine

PILIER 3 — STYLE MATCH AR
  Le produit s'adapte à la déco du client en temps réel
  4 ambiances → couleurs harmonisées → morphing fluide

PILIER 4 — DASHBOARD SAAS
  Gestion complète depuis web ou mobile
  Stats, QR codes, embed codes, analytics
```

## Le parcours en 3 étapes
1. Le commerçant filme son produit avec son iPhone → guide AR automatique
2. ARShot génère un modèle 3D photoréaliste en 2-3 minutes
3. Il reçoit un lien + QR code shareable Instagram/TikTok/site/flyers

## Le client final
Scanne le QR code → Safari/Chrome s'ouvre → le produit apparaît dans son salon en AR. Zéro app. Zéro friction.

## Pipelines de génération 3D

```
PIPELINE A — iPhone (70% des cas)
  Coût : 0€ (on-device)
  Temps : 2-3 min
  iPhone → Apple Object Capture RealityKit
  → USDZ natif → Upload R2 → QR + lien

PIPELINE B — Android/Fallback (25% des cas)
  Coût : 0.008€/modèle
  Temps : 45s GPU
  Photos → FlashVDM (Hunyuan3D-2 Turbo) via Modal Labs
  → 5 steps DiT Sampling (Progressive Flow Distillation)
  → 0.49s VAE Decoding (Hierarchical Volume Decoding)
  → Mesh Extraction (Marching cubes)
  → GLB → converti USDZ → Upload R2

PIPELINE C — Objets complexes (5% des cas)
  Coût : 0.03€/modèle
  Temps : 10s RTX 4090
  Vidéo → NeRF → Gaussian Splatting hybride
  → Transparences, réflexions, cas difficiles

ORCHESTRATEUR (déjà dans le backend)
  Choix automatique selon : device, disponibilité API, qualité capture
  Fallback en cascade si échec

POST-PROCESSING (sur tous les pipelines)
  → Compact3D compression (VQ 10× = 200MB → 20MB)
  → SuraGS pruning (75% gaussiens supprimés, 65% GPU en moins)
  → LOD Generation (3 niveaux : mobile/desktop/preview)
  → Export SPZ + GLB + USDZ
```

## Cache Service Worker (scans gratuits)
```javascript
// Modèles 3D cachés après premier téléchargement
self.addEventListener('fetch', (event) => {
  if (event.request.url.includes('.spz')) {
    event.respondWith(
      caches.match(event.request).then((cached) => {
        if (cached) return cached; // 0€ bandwidth
        return fetch(event.request).then((response) => {
          caches.open('ar-models-v1').then(cache => {
            cache.put(event.request, response.clone());
          });
          return response;
        });
      })
    );
  }
});
```
